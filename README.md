# Framework Salt

Bem-vindo ao repositório de documentação do **Framework Interno ERP Salt**.

Este repositório contém a infraestrutura e o guia completo para criação de módulos CRUD, componentes dinâmicos, e integrações Livewire/Filament no ecossistema da Sagatech.

📚 **Para ler a documentação completa, acesse o nosso site oficial gerado automaticamente:**
👉 **[Ler a Documentação Oficial (VitePress)](https://sagatechdev.github.io/framework-salt/)**

---

### Onde estão os arquivos?

A documentação interativa não fica neste arquivo Raiz. Todo o código dos guias está encapsulado de forma modular dentro da pasta **[`docs/`](./docs)** para ser compilado limpo e organizado pelo VitePress. 

*   [Visite o `docs/index` para editar a página principal.](./docs/index.md)
*   [Visite `docs/resource` para editar sobre Base do CRUD.](./docs/resource.md)

### Como rodar o site localmente para escrever:

Caso você queira editar este projeto e ver as páginas VitePress atualizando em tempo real no seu navegador:

```bash
npm install
npm run docs:dev
```
Acesse `http://localhost:5173/framework-salt/` no seu Browser.
