# Componentes de Formulário & Layout

O framework disponibiliza uma série de componentes avançados orientados a objetos. Eles foram criados para interagir perfeitamente de maneira declarativa com as nossas classes de páginas Livewire.

## Componentes Disponíveis

### Entradas (Inputs)
| Componente | Descrição |
| ---------- | --------- |
| **[Money](Money.md)** | Campo de entrada com máscara otimizada para valores monetários (moeda). Garante a formatação correta. |
| **[DateRange](DateRange.md)** | Encapsula dois campos de data, permitindo configurar nativamente `data de início` e `data de término`. |
| **[FileUpload](FileUpload.md)** | Integra com o FilePond nativamente, controlando tipos de arquivos aceitos e limites de upload. |

### Componentes de Repetição (Layout)
| Componente | Descrição |
| ---------- | --------- |
| **[Repeater](Repeater.md)** | Permite duplicar blocos de componentes como um array. |
| **[RepeaterContainer](../RepeaterContainer.md)** | Uma evolução do Repeater padrão para trabalhar com wildcards em arrays de configurações complexas. |
| **[CollapsibleGroup](CollapsibleGroup.md)** | Agrupador retrátil de repetições. |
